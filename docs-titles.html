Правила присуждения титулов по набору сертификатов, полученных на монопородных выставках

Для получения титула Чемпион НКП “Немецкий шпиц” необходимо предоставить сертификаты в одной из комбинаций, перечисленных ниже:

  • 1 сертификат ЧК;

  • 2 сертификата ПК;

  • 1 сертификат ПК + 2 сертификата КЧК;

  • 4 сертификата КЧК;

  • 1 диплом ЮЧК + 1 сертификат ПК;

  • 1 диплом ЮЧК + 2 сертификата КЧК.

При этом однократно КЧК может быть засчитан путем набора 3-х сертификатов СС.

Титул Чемпион НКП “Немецкий шпиц” присуждается однократно.

Для получения титула Юный Чемпион НКП “Немецкий шпиц” необходимо предоставить сертификаты в одной из комбинаций, перечисленных ниже.

  • 1 сертификат ЮЧНКП;

  • 2 сертификата ЮПК;

  • 1 сертификата ЮПК + 1 сертификат ЮКЧК;

  • 3 сертификата ЮКЧК.

При этом однократно ЮКЧК может быть засчитан путем набора 3-х сертификатов ЮСС.

Титул Юный Чемпион НКП “Немецкий шпиц” присуждается однократно.

 

Для получения титула Чемпион Ветеран НКП “Немецкий шпиц” необходимо предоставить дипломы/сертификаты в одной из комбинаций, перечисленных ниже.

  • Диплом Чемпиона* + 1 сертификат Победитель класса (CW) ветеранов, полученный на выставке ранга ЧК.

  • Диплом Чемпиона* + 2 сертификата Победитель класса (CW) ветеранов, полученных на выставках ранга ПК.

  • Диплом Чемпиона* + 1 сертификат Победитель класса (CW) ветеранов, полученный на выставке ранга ПК, + 1 сертификат Победитель класса (CW) ветеранов, полученный на выставке ранга КЧК.

  • Диплом Чемпиона* + 3 сертификата Победитель класса (CW) ветеранов, полученных на выставках ранга КЧК.

*Диплом Чемпиона - диплом Чемпиона страны (признаваемый в системе FCI) или диплом  Чемпион Национального Клуба Породы страны.

Титул Чемпион Ветеран НКП “Немецкий шпиц” присуждается однократно.

Титул Чемпион Ветеран НКП “Немецкий шпиц” не вносится в родословные РКФ.

Для получения титула Абсолютный Чемпион НКП “Немецкий шпиц” необходимо предоставить сертификаты в одной из комбинаций, перечисленных ниже.

  • Диплом ЧК + 2 диплома с титулами ЛК/ЛС, полученных на выставках ранга ПК + сертификаты с результатами тестирования;

  • Диплом ЧК + 1 диплом с титулом ЛК/ЛС, полученный на выставке ранга ЧК + сертификаты с результатами тестирования;

  • Диплом ЧК, полученный на выставке ранга ЧК/диплом Чемпион национальной выставки породы + 1 диплом с титулом ЛК/ЛС, полученный на выставке ранга ЧК или ПК + сертификаты с результатами тестирования;

Составляющие титула Абсолютный Чемпион НКП “Немецкий шпиц” не могут быть получены в один день.

Под “Сертификатами с результатами тестирования” подразумеваются следующие сертификаты: сертификат с результатами исследования  люксации пателлы (для всех разновидностей) + сертификаты с результатами исследования дисплазии тазобедренных и локтевых суставов (для больших шпицев и вольфшпицев).

Титул Абсолютный Чемпион НКП “Немецкий шпиц” присуждается однократно.

Титул Абсолютный Чемпион НКП “Немецкий шпиц” не вносится в родословные РКФ.

 

Все сертификаты, предоставленные для получения какого-либо из титулов НКП, должны быть получены у разных судей.

Для собак, принадлежащих иностранным гражданам, действуют те же правила получения титулов.
