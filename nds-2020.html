Оценка коленных суставов (PL)

В рамках Чемпионата породы - 2020 будет организована клиническая оценка коленных суставов (PL) с выдачей сертификата установленного образца.

Для проведения оценки будут необходимы

  • собака старше 12 месяцев (возможна предварительная оценка коленных суставов собаки старше 7 месяцев),
  • заполненный договор-заявка
  • оригинал родословной,
  • читаемые клеймо и/или чип.

Оценку будет производить ветеринарный врач Мезин Андрей Владимирович.

Стоимость оценки по предварительной записи - 1100 рублей для первой собаки одного владельца, 1000 рублей - для каждой последующей собаки одного владельца.
